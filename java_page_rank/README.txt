Java page rank

In this program i emplamented googles page rank algorithm in java 8. The program reads in a line at a time from a file. It takes out all the stop words and punctuation. It then puts each word into a dictionary with a value of one or if the words already in the dictionary add to the value 1. The program links all lines to others with the same repeating words in both. Lastly, it performs the algorithm on each line to create its page rank.
